## Install

```bash

npm i -g clipcopy





## Watch clipboard (Ctrl+C to stop)

clipcopy watch



## List the last 20 items

clipcopy list



## Search for a term

clipcopy search npm



## Copy item at index back to clipboard

clipcopy 2

## or

clipcopy copy 2



## Set history limit

clipcopy limit 100



## Clear history

clipcopy clear
